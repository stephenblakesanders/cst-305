To run the program:
	1) Install the project7.py and project7-part2.py files
	2) Open using an IDE of your choice
		* In class we use PyCharm
	3) Install Librareis numpy, scipy, matplotlib, and simpy
	4) Click on the "Run" icon in the IDE

When program is ran:
	Part 1 will ask for user input for sigma, rho and beta values and then graph the model in three dimensions
	Part 2 will demonstrate the calculations of Utilization, Throughput, E[N] and E[T] with changing lambda and mu values
