To run the program:
	1) Install the final-part1.py and final-part2.py files
	2) Open using an IDE of your choice
		* In class we use PyCharm
	3) Install Librareis numpy, scipy, matplotlib, and simpy
	4) Click on the "Run" icon in the IDE

When program is ran:
	Part 1 will ask for user input for N iterations, allowing the riemann sum to be calculated
	Part 2 will demonstrate the calculations of the time it took to download files, as well as show and calculate the riemman sum for them
